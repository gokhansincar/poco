Units of measurement in CSS

Relative units

em

Represents the inherited font size of the nearest parent element.
The line-height, font-size, margin-bottom, and margin-top CSS properties are often expressed in em.
We read it "M's" as it cames from traditional typography letter "M". 


		html {
			font-size: 20px;
		}

		html p {
			width: 20em;		/* => 400px; */
			font-size: 1em;		/* => 20px */
			line-height: 1.5em;	/* => 30px */

			margin-top: 1em;	/* => 20px; */
			padding: 2em;		/* => 40px around the text; */
		}

The code above will give us:

Lorem Ipsum M

rem (root-em)

represents the font-size of the root element (wich is the html element)


		html {
			font-size: 20px;
		}

		div {
			font-size: 12px;
		}

		div p {
			font-size: 1rem;	=> 20px
		}

Proportional units

%

Relative to the size of the parent element


		div {
			width: 800px;
		}
		div p {
			width: 50%;		=> 400px
		}

Absolute units

px

For on-screen display, typically corresponds to one pixel of the display
For high-resolution screens and printers, a CSS pixel corresponds to several pixels of the device, so the number of pixels per inch (ppi) stays around 96 (!) 


		div 1 {
			width: 200px;
			height: 50px;
		}
		div 2 {
			width: 800px;
			height: 50px;
		}
