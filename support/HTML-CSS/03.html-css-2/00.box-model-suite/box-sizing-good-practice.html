box-sizing, best practice

Code to put at the top of your styles (after reset or normalize)


				html {
					box-sizing: border-box;
				}

				*, *:before, *:after {
					box-sizing: inherit;
				}

Code to put on a html element where the box-sizing is different


				element-html-cible, element-html-cible * {
					box-sizing: content-box;
				}
