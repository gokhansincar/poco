CSS - The display property

The display property defines the type of box used for rendering an element.

display: block

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

BY DEFAULT...

  • I have a width of "100%"
  • I have a height of "auto" (text line height)
  • I'm invisible without content
  • common tags: article, aside, div, figure, footer, h1(etc), header, hr, main, nav, p, ul, section, table (see all)
I am a text in a block box

display: inline

An inline element does not start on a new line and only takes up as much width as necessary.

BY DEFAULT...

  • I have a width "auto" (width of the text)
  • I have a height "auto" (text height)
  • I am invisible without content
  • common tags: a, b, cite, code, em, img, span, strong (see all)

I am a text in a paragraph. Me too, but in a span tag. Often used to highlight a word, a link, or to display an image like this...

display: inline-block

A box element that "flows" into the surrounding content as if it were an inline element. A hybrid between a block and inline element ...

Online too but I can have a padding and a "margin", rather cool.

display: flex

The flex CSS property sets how a flex item will grow or shrink to fit the space available in its flex container.
