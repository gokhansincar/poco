CSS - filter

The filter CSS property applies graphical effects like blur or color shift to an element.
Filters are commonly used to adjust the rendering of images, backgrounds, and borders.

filter: filter-function [filter-function]* | none

Examples


					#filter-1 {
						filter: drop-shadow(0 0 20px blue);
					}

					#filter-2 {
						filter: blur(3px);
					}

					#filter-3 {
						filter: grayscale(80%);
					}

					#filter-4 {
						filter: hue-rotate(90deg);
					}

					#filter-5 {
						filter: opacity(25%);
					}

					#filter-6 {
						filter: sepia(100%) brightness(0.7);
					}

Compatibility with old browsers


						#filter-2 {
							-webkit-filter: blur(3px);
							-moz-filter: blur(3px);
							-ms-filter: blur(3px);
							-o-filter: blur(3px);
							filter: progid: DXImageTransform.Microsoft.Blur(PixelRadius = '3');
							filter: blur(3px);
						}
