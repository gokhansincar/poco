CSS - Position

The position CSS property sets how an element is positioned in a document.

The top, right, bottom, and left properties determine the final location of positioned elements.

position: static;

The element is positioned according to the normal flow of the document.

The top, right, bottom, left, and z-index properties have no effect. This is the default value.

position: relative;

The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left.

The offset does not affect the position of any other elements; thus, the space given for the element in the page layout is the same as if position were static.


					#c2-two {
						position: relative;
						top: 30px;
						left: 30px;
					}

position: absolute;

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any;

In this example, the parent (#c3) is in "relative" position and (therefore) the children in absolute position are positioned relative to him.


					#c3 {
						position: relative;
					}

					#c3-two {
						position: absolute;
						top: -20px;
						left: -20px;
					}

position: fixed;

Fixed positioning is similar to absolute positioning except that here the "parent" reference is the viewport (browser window).

This positioning is often used to create a floating element that stays in the same position, even when scrolling!

In the following example, the "two" box is set at 20 pixels from the top of the page (viewport) and 20 pixels from the right edge.


					#c4-two {
						position: fixed;
						top: 20px;
						right: 20px;
					}
