CSS - z-index

The z-index property defines the "z-order" of an element positioned in absolute relative to other elements.

When elements overlap, the z-order determines the order of the different layers that will form the elements.

An element will cover another element if its z-index value is greater than that of the second element.

One can make an analogy with layers in Photoshop.

Example

Below is an example of positioning with z-index.

The divs #one and #three have a z-index of 2

The divs #two and #four have a z-index of 1.


					#container .box {
						position: absolute;
					}

					#one {
						top: 60px;
						left: 30px;
						z-index: 2;
					}

					#two {
						top: 30px;
						left: 100px;
						z-index: 1;
					}

					#three {
						top: 60px;
						left: 170px;
						z-index: 2;
					}

					#four {
						top: 30px;
						left: 240px;
						z-index: 1;
					}
