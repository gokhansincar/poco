# JAVASCRIPT - DATA TYPES


## Primitives
* **Number**		(`25, 23.2`)
* **String**		(`"Hello World!"`, `'My name is Bob'`)
* **Boolean**		(`true`, `false`)
* **Null**			(the value is 'nothing' → `var bob = null;`)
* **Undefined**	(the value is not been set → `var bob;`)

## Objects
* **{ }** (arrays, functions)
